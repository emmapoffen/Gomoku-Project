CSCE 390: Objects & Design: Two Player Networked Game of Gomoku - Iteration 2
To run the application:
1. Run the Gomoku Server jar file, Iteration3Server.jar
2. Enter desired port number into text field, or leave it as the default,
    and press Start. This starts the server.
3. Run the Client jar file, Iteration3Client.jar
4. Enter desired server and port information into appropriate text fields, or
    leave the defaults. Enter other desired information and proceed with testing.
